How it all started

January 06 2022

After weeks of peruasion, my brother Nick convinced me to start a NFT project. The original idea was to make a game just to learn about NFT's myself and being a masters Computer Science with specialization in Data Science, i thought it would be fun to play around with creating NFT data and passing it through some Machine Learning models I was learning in class.
